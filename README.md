# Finance Tracker App — Hafta 1

## Bu Hafta: Kurulum + Giris/Kayit Ekrani

### Kurulum
```bash
npm install
npx expo start --ios
```

### Backend
```bash
cd FinanceWebAPI
dotnet run
```
> Backend portunu `src/constants/api.js` dosyasindaki `BASE_URL`'e yaz.

### Ekran Durumu
- ✅ AuthScreen — Giris/Kayit
- 🔨 HomeScreen — Ana Sayfa (daha yapilmadi)
- 🔨 TransactionsScreen — Islemler (daha yapilmadi)
- 🔨 SubscriptionsScreen — Abonelikler (daha yapilmadi)
- 🔨 GoalsScreen — Hedefler (daha yapilmadi)
